# MDynamics
Physics Simulator

A Mayami Physics Simulator.
<br />
You can simulate rigid bodies and soft bodies!
<br />

### Features
* Customizable Shapes
* Compatiable with All graphics libraries
* Rigid Body
* Soft Body
